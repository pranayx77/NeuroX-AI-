# NeuroX AI — Telegram Community Automation

A single-page landing page for **NeuroX AI**, a Telegram bot service that provides AI-powered automated replies in discussion groups.

## Features

- 🎨 **Dark orange theme** with CSS custom properties
- ✨ **WebGL Aurora** background with custom GLSL shaders
- 🧭 **Hamburger card navigation** with GSAP animations
- 🃏 **3D tilt cards** with mouse-tracking perspective
- 🧲 **Magnetic cursor** effect with glow follow
- 📜 **Scroll reveal** animations via IntersectionObserver
- 🌊 **Floating particles** and parallax orbs
- 📱 **Responsive** design (breakpoint at 900px)

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, gradients, animations
- **Vanilla JavaScript** — Zero framework dependencies
- **GSAP 3.12.5** — Animation library
- **WebGL2** — Custom shaders for aurora effect
- **Google Fonts** — Bebas Neue + DM Sans

## Structure

index.html          # Single-file landing page (all CSS/JS inline)
