Under The Hood

SIMPLE BY DESIGN

The entire flow is clean. Listener → Filter → AI → Reply. That's the whole system.

neurox_bot.js
// NeuroX AI — Core auto-reply loop
bot.on('message', async (ctx) => {
  if (!shouldReply(ctx)) return; // smart filter — skips spam, bots, short msgs
 
  const config = await getChannelConfig(ctx.chat.id); // your custom prompt
  const reply = await askAI(ctx.message.text, config.systemPrompt);
 
  await ctx.reply(reply, {
    reply_to_message_id: ctx.message.message_id // threaded reply
  });
 
  logReply({ chat_id: ctx.chat.id, reply }); // saves to analytics
});
